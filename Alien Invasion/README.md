  Download these and just run the python file of alien_invasion is OK.  
  Pressing the right or left arrow key can control the ship to move right or left.  
  Pressing the shift key can restore shield if your shield is broken.  
  You can change the settings in class Settings which involved the speed of the ship, alien's bullets and ship's bullets and the range of the alien's speed and alien's shooting time repetition.  
  Note that the shield can only defend the attack from the alien bullet but cannot defend the attack from the alien ships crashing.
